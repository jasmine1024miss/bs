## Kylin/openKylin 下一步

复制项目到 Kylin/openKylin 后：

```bash
bash scripts/check_kylin_compat.sh
SAFEOPS_MODE=real bash scripts/start_backend_kylin.sh
bash scripts/start_frontend_kylin.sh
```

重点检查：

- `/api/environment/probe` 中 `effective_mode` 是否为 `real`。
- `systemctl` 是否可用。
- `journalctl` 是否可用。
- `ss` 是否可用。
- `ps` 是否可用。
- Nginx 故障场景是否能复现。
